
# AliGenie Demo

基于 [AliGenie 开发者平台](http://doc-bot.tmall.com/) 实现的天猫精灵的扩展应用项目。项目 fork 自[aligenie_calc](https://github.com/joans321/aligenie_calc)。

目前计划是实现让天猫精灵支持个人网易云音乐歌单的智能播放。现在流程有一点还不太确定，关于让天猫精灵播放歌曲要返回怎样的格式。对这个点感兴趣的可以一起开发。

# Usage

~~~sh
# 安装依赖环境
yarn install

# 启动服务
yarn start

# 执行测试
yarn test
~~~
